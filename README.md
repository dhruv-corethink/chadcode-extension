# ChadCode

> AI coding assistant powered by Corethink for planning, building, and fixing code.

- Generate code from natural language
- Checks its own work
- Run terminal commands
- Automate the browser
- Latest AI models
- API keys optional

## Key Features

- **Code Generation:** Generate code using natural language descriptions
- **Task Automation:** Automate repetitive coding tasks
- **Automated Refactoring:** Refactor and improve existing code
- **MCP Server Support:** Extend agent capabilities with MCP servers
- **Multiple Modes:** Plan with Architect, Code with Coder, Debug with Debugger, or create custom modes

## Getting Started

1. Install the ChadCode extension
2. Connect to api.corethink.ai to access AI models
3. Start coding with AI that adapts to your workflow
